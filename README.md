A Guide to Turn a 4056 Server Fan INTO A POWERFUL HANDHELD FAN!!

# Features
The 4056 model lets for a much higher energy density, its counter rotating feature directs the airflow to be straight and improve static pressure
This fan has tens of thousands of rated usage, it can last way way longer than even the batteries (which already has a long lifetime)
The knob lets you adjust the speed based on your needs.
The type C charger lets you charge with your phone charger

# 3D printing things: 
- Print with the orientation the same as you see on the file!
- All printed using Black ASA/ABS
- All 0.4mm Nozzle
- All 0.2/0.16/0.12 Layer height
- All 40% infill
- All 6 or more Walls Perimeter
- Slow print speed
- Fan at 0%
- 50 degrees C enclosure (if you have)
